# Task-Tracker-CLI

## Description

**Taskly** is a lightweight command-line interface (CLI) application for efficient task management. The idea for the project was taken from [task-tracker](https://roadmap.sh/projects/task-tracker) challenge from [roadmap.sh](https://roadmap.sh/).

## Usage

```bash
$ task-cli add description

$ task-cli update id [description] 

$ task-cli mark-done id

$ task-cli mark-in-progress id

$ task-cli delete id

$ task-cli list [{done,in-progress,todo}]
```
